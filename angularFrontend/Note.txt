Steps how to create, clone, and add files in your repository.

Step 1: Create a repository in your github account.
Step 2: Clone your repository from github to your local device. (Using cmd: git clone <url>).
Step 3: Now create a angular Project. (Using cmd: ng new <Project_name>).
Step 4: Now add your all files in your remote repository. (Using cmd: git add .)
Step 5: Commit all things that reflect in your remote repository. (Using cmd: git commit -m " messege ")
Step 6: before pushing your code check in which branch you are Commited (Using cmd: git branch)
        if you want to switch the branch use (git branch -M main). Delete old Master: (git push origin --delete master)
Step 7: Now push your code (Using : git push -u origin main)
